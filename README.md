### <center> Project 1 - RGB lights blinking using millis() </center>

* 1 Non-Blocking Timing: Replaces delay() with millis(), allowing the CPU to run multiple tasks at once.
* 2 Independent Arrays: Manages Pins, Intervals, and Timestamps in parallel arrays for organized, scalable control.
* 3 Stopwatch Comparison: Subtracts the "last recorded time" from the "current time" to check if an interval has passed.
* 4 State Flipping: Uses the ! (NOT) operator to toggle LED states instantly between HIGH and LOW.
* 5 Single-Loop Refresh: Captures now once per cycle to ensure all LEDs are evaluated against the same clock pulse.
